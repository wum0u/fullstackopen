# 0.6: New note in Single page app diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    activate browser
        Note right of browser: Browser creates a note object
        Note right of browser: Then appends it to existing list of notes
        Note right of browser: The input field gets cleared to be empty
        Note right of browser: And finally calls the redraw function to render new note onto the page
    deactivate browser

    browser ->> server: POST: https://studies.cs.helsinki.fi/exampleapp/new_note_spa
        Note right of browser: The user's note content and timestamp is sent as a JSON object as the payload to the server
    activate server
    server -->> browser: 201 Response 
        Note left of server: Server responds with JSON {"message": "note created"} to denote the note was successfully created
    deactivate server
```
